# Receep.io

Receep.io is a simple and professional platform for creating, sending, and organizing digital receipts.

## Features
- Create digital receipts in seconds  
- Send receipts via link or PDF  
- Recipient digital acceptance  
- Organized dashboard for payments and collections  
- Clean and modern interface
